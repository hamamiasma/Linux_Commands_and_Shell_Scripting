# Informational, Navigational, & Management Commands


#  مشروع تدريبي: إدارة الملفات والمجلدات في لينُكس

##  الهدف
تعلم واستخدام أوامر لينُكس الأساسية الخاصة بإدارة الملفات والمجلدات، من خلال تنفيذ سيناريو عملي واقعي.

---

## خطوات المشروع

###  1. إنشاء مجلد المشروع والدخول إليه:
```bash
mkdir MyProject
cd MyProject
```

###  2. إنشاء هيكل المجلدات الفرعية:
```bash
mkdir scripts notes output
```

🔖 النتيجة:
```
MyProject/
├── scripts/
├── notes/
└── output/
```

---

###  3. إنشاء ملفات أساسية داخل المشروع:
```bash
touch scripts/run.sh notes/todo.txt output/results.txt
```

---

###  4. عرض هيكل المشروع الكامل:
```bash
ls -R
```

---

###  5. تعديل سكربت التشغيل:
افتح الملف باستخدام `nano`:
```bash
nano scripts/run.sh
```

📋 أضف السطور التالية:
```bash
#!/bin/bash
echo "مرحباً بك في مشروع لينيكس!"
```

📝 للحفظ والخروج:
- اضغط `Ctrl + O` → ثم `Enter` للحفظ  
- ثم `Ctrl + X` للخروج

---

###  6. تجربة تشغيل السكربت (سيفشل حالياً):
```bash
./scripts/run.sh
```
❌ النتيجة:  
```bash
Permission denied
```

---

###  7. إعطاء صلاحية تنفيذ للسكربت:
```bash
chmod +x scripts/run.sh
./scripts/run.sh
```

✔️ النتيجة:
```
مرحباً بك في مشروع لينيكس!
```

---

###  8. نسخ ملف الملاحظات كنسخة احتياطية:
```bash
cp notes/todo.txt notes/todo_backup.txt
```

---

###  9. إعادة تسمية ملف النتائج:
```bash
mv output/results.txt output/final_results.txt
```

---

###  10. حذف ملف النسخة الاحتياطية:
```bash
rm notes/todo_backup.txt
```

---

###  11. حذف مجلد فارغ:
```bash
rmdir output
```

> 📌 ملاحظة: يجب حذف جميع الملفات داخل المجلد قبل حذفه.

---

##  الأوامر المستخدمة

| الأمر | الوظيفة |
|-------|----------|
| `mkdir` | إنشاء مجلد |
| `cd` | الانتقال بين المجلدات |
| `touch` | إنشاء ملف فارغ |
| `nano` | تحرير الملفات النصية |
| `chmod` | تعديل صلاحيات الملفات |
| `ls -R` | عرض جميع الملفات والمجلدات بشكل متسلسل |
| `cp` | نسخ الملفات |
| `mv` | إعادة تسمية أو نقل الملفات |
| `rm` | حذف الملفات |
| `rmdir` | حذف المجلدات الفارغة |

---

## ✅ النتيجة النهائية

- بناء هيكل مشروع
- إنشاء وتعديل الملفات
- التعامل مع الصلاحيات
- نسخ، نقل، وحذف الملفات والمجلدات
